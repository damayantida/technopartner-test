# 🧾 Tes Tahap 2 Technopartner

## 🚀 Features

- 🔐 Login screen with email & password
- 🎨 Custom UI with NativeWind (Tailwind for React Native)

  - Saldo & Points info
  - QR Code
  - Partner logo section

- **Expo** (React Native framework)

- **NativeWind** (TailwindCSS for RN)
